# -react-essentials-assignment

This repo has two small projects I made while learning React.

## Projects

**Portfolio Card**

* Simple profile card with skills
* Has theme toggle, image change, and like button

**Movie Explorer**

* Search movies
* Shows results as you type
* Can add/remove favourites

## Setup

Clone the repo and run:

```bash
git clone https://github.com/studiesmail/-react-essentials-assignment/
npm install
npm start
```
